PL-2303 Windows CE driver Release Note v1.0 (2002/06/12)
===========================================


To install PL-2303 driver on Windows CE:
---------------------------------------
1. Transfer 'USBSER.ARM.CAB' to ARM WinCE platform.
2. Double Click (Execute)'USBSER.ARM' to install driver.
3. Plugin PL-2303 cable.
4. Software Reset the platform.
5. Execute 'ComNo.exe' to know the port number.


To remove PL-2303 driver on Windows CE:
--------------------------------------
1. Unplug PL-2303 cable.
2. Double Click (Execute) 'RegKiller.exe'.
3. Software Reset the platform.
4. Remove 'Prolific PL-2303 Driver' in Remove Programs of Control Panel.


Utilities:
---------
1. RegKiller.exe - Delete the part of registry of PL-2303 driver.
2. ComNo.exe - display the PL-2303 virtual COM port number.
